Lab 07

Animação

1. HTML:
a. Adicione uma linha na tabela para a página 'Projetos.html' com o nome Animação e o link para a página
b. Crie uma nova página chamada 'Animation.html'
c. Na nova página, utilize a tag canvas para criar a área de desenho
d. Configure o canvas para ter tamanho 300x300
2. CSS:
a. Adicione cor do fundo
b. Centralize os elementos na página
3. JavaScript:
a. Utilizando JS, faça o controle de um quadrado por meio da posição do ponteiro do mouse.
O quadrado vermelho deve seguir o ponteiro do mouse.
b. O quadrado vermelho não pode, jamais, desaparecer do canvas
